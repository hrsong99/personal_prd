Problem Solution

02 노후화된 입력 항목 등록 플로우 재설계
불필요하거나 지엽적인 옵션 항목이 포함되어 데이터 관리와 사용자 불필요하거나 중복되는 항목은 정리하고

입력 과정의 사용성이 떨어지는 문제가 있었습니다. 등록 흐름을 필수적인 것 중심으로 단순화하였습니다.

03 법적 필수 항목 누락 법적 필수 항목 반영
필수 입력 26개 항목의 필드 자체가 없거나 선택 입력 사항으로
광고 표시법에 해당하는 입력 항목을 모두 반영하고

설계되어 있어 사용자가 법적 리스크에 무방비하게 노출되었습니다. 필수 입력값으로 구조화하여 법적 리스크를 해소하였습니다.

04 모바일 환경 미지원 모바일 최적화 설계
PC 환경만 지원하여 매물 수정이나 광고 중단 등
PC/모바일 어느 환경에서든 사용자의 목적에 맞게

광고 표시법과 연관된 업무에 대해 즉각 대응이 어려웠습니다. 매물 관리를 할 수 있도록 서비스를 재설계 하였습니다.
