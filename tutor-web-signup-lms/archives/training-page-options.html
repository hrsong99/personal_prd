3가지 방향

세 옵션 모두 공통으로 적용한 변화:
  • 실제 PODO 디자인(P○DO 로고, 좌측 네비, 라임 그린 액티브 상태)에 맞춤
  • 코스마다 예상 소요시간을 눈에 띄게 표시 ("약 25분")
  • 상단에 전체 진도 + 남은 시간 요약 — "할 만하다"는 느낌 강조
  • 온보딩을 교육 흐름 안의 마지막 단계로 항상 보이게 (잠금 상태 → 활성화 상태 전환)
  • 코스마다 아이콘 + 컬러로 시각적 구분 (필수=따뜻한 톤, 보충=쿨한 톤)

👍 한눈에 분량 파악 · 시각적으로 가장 가볍고 깔끔 · 카드가 독립 단위라 진도 추적 직관적

한 줄 요약

A · 아이콘 그리드
레퍼런스 이미지에 가장 가까움. 카드 그리드로 한눈에 분량 파악. 깔끔하고 가벼움.

추천 케이스: 코스가 3~6개 정도, 진행 순서를 강제하기보다 "내 페이스로" 진행하게 하고 싶을 때.
B · 여정 헤더 + 큰 리스트
상단의 3-스텝 여정이 "내 위치 + 끝까지의 거리"를 가장 명확히 보여줌. 정보 밀도가 가장 높음.

추천 케이스: 다음 단계(온보딩 → 수업 시작)까지의 흐름을 사용자에게 가장 강하게 각인시키고 싶을 때.
C · 세로 로드맵
학습 앱 스타일의 타임라인. "끝까지의 줄"이 시각적으로 보여 가장 "doable"한 느낌. 모바일에서 강함.

추천 케이스: 진행 순서를 엄격히 강제하고, 한 번에 하나씩 끝내는 흐름을 만들고 싶을 때.
3가지 옵션 — 화면 목업. 실제 디자인 시스템(컬러·타이포)에 맞춰 최종 조정 필요.
